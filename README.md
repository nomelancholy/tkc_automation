# tkc_automation

## spec

- pythono
- selenium
- github action

## step

1. upload index, artist, title, featuring, link, link_type 을 넣어 github issue 생성 후 셀프 어사인 (이동중 핸드폰으로도 업로드할 수 있게 하기 위해)

2. github action이 assign에 맞춰 돌며 업로드
